Wireframe Document

CAPA AI — Full App Wireframes

Structure and flow wireframes for the reworked CAPA AI application. Grayscale, structure-only. Focus on layout, information architecture, and UX flows for the 8D CAPA workflow — the core user journey.

Core UX challenge: CAPA is a compliance-critical 7-step workflow (8D) used by pharmaceutical QA teams. The current app has no sidebar navigation, confusing redundant entry points, and an AI assistant (Nova) that appears in 4 inconsistent patterns. This wireframe reworks the app around a persistent sidebar, a CAPA Hub with 8D progress panel, and a unified Nova AI pattern.

App Structure & Navigation

Full sitemap with primary surfaces and their sub-routes.

/dashboard ← Authenticated shell (sidebar + topbar) │ ├── / ← My Work / Inbox (default landing) │ ├── /analytics ← Dashboard (KPIs, charts, heatmap) │ ├── /findings ← Findings list │ ├── / ← Table + filters + slide-over detail │ └── /[id] ← Finding full page (optional) │ ├── /capa ← CAPA management │ ├── / ← CAPA List (all CAPAs, filterable) │ ├── /new ← ★ New CAPA Intake wizard (4 steps) │ ├── /[id] ← ★ CAPA Hub (overview + 8D progress panel) │ └── /[id]/8d │ ├── /problem ← D1 Problem Statement │ ├── /containment ← D2 Containment │ ├── /rca ← D3 Root Cause Analysis │ ├── /ca ← D4 Corrective Action │ ├── /pa ← D5 Preventive Action │ ├── /verification ← D6 Verification │ └── /signoff ← D7 Sign-Off & Closure │ ├── /reports ← Reports & Analytics (future) │ └── /settings ← Settings (future) ├── /profile ← Personal profile ├── /users ← User & role management (RBAC) └── /integrations ← EQMS / API config

Navigation Principles

  • Sidebar (persistent) — primary nav on desktop. 6 items: My Work, Dashboard, Findings, CAPAs, Reports, Settings. Always visible.
  • Top bar — breadcrumb + notification bell + Nova AI button + user avatar. No primary actions here.
  • My Work is the default landing — not Dashboard. QA Analysts need "what do I work on today", not charts. Dashboard is for Managers.
  • 8D steps live inside CAPA Hub — left progress panel always shows D1–D7 with current step highlighted. Click any step to navigate. No need to go back to CAPA detail first.

  • Card grid: 2-column. Each card shows CAPA ID, Source ID, similarity %, root cause, corrective action, outcome badge, type badge, year.
  • Detail modal: Full breakdown for one historical case. Read-only. Includes Nova disclaimer.
